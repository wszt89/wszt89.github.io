路虽远行则将至，事虽难做则必成！

发表于 2019-03-24   |   分类于 总结   |     |   阅读次数

在大家的同心协力下，系统成功上线了，但是开心不起来。写个晚到的总结，顺便思考一下人生。

工作

自17年末加入自动化，搞过的自动化设备有：

  • 机械臂
  • 立库
  • 输送线
  • 无人叉车

并不会直接和硬件打交道，更多的是想怎么把更多的任务喂给设备。业务上做了：

  • 机械臂拣选
  • 立库直出
  • 立库 + 机械臂 + 贴标 全无人出库
  • 预包
  • 半托 + 效期出库
  • 叉车搬运

经过双十一的洗礼，慢慢的我们受到了关注。快速的业务迭代，也让我们的系统变得不那么优雅。于是S2进行了彻底地重构。

开心的事情：

和很多同事越来越熟，打打球、喝喝酒，好像也没有那么的孤单。

认识了一些朋友，双十一的大考快过去了，几个人喝着酒聊着彼此的故事；打着扑克、嗑着瓜子，最后给大家唱《送别》，真希望大家只是朋友、不是同事。

在现场发现的问题一一解决：任务的串联、通过叉车搬运等等，能落下去的感觉真好。

无奈的事情：

有不喜欢的同事，老板却很是喜欢：不需要解决工单，系统出问题了也优哉游哉，各种成果还可以往她身上挂。

有点僵的同事：有些时候自己的方案确实可能会动了别人的奶酪，坚持了自己的方案，失去了一些和气。

思考的事情：

周末在家，仓库有问题钉对应的同事没反应，刚想生气，突然想到：也许错的是自己。为什么周末还要解决这些问题？难道自己这样慌慌张张的状态是对的吗？为什么不是去陪家人散步聊天？

我们能做的事情别人能不能做的了甚至做的更好？在现在的团队学到的东西是什么？能积累到的又是什么？

未来的事情：

从各个角度来看，自动化都应该是未来中国非常有希望的一环，现有自动化中的各个环节是什么样的？如何才能成为这个领域的专家？或者说如何做到比别人更懂自动化？

生活

小孩长得飞快，慢慢地开始学会说话。有时候会傻傻地想：

我愿意为他们付出一切，没有一点犹豫（有点肉麻哈）。

然而现实中的我，却是周末趴在电脑前，晚上趴在电脑前。像是没了我仓库就玩不转了，又或者像是在拯救地球。我们一起熬着夜、拼着命，把一个一个的项目做上去，看起来能做很多事情，但：

我们没办法让时间停留，一秒都不行，过去了就是过去了，再也不会回来。

错过的东西太多了，没有在莺飞草长的时节感受春风，没有去夏天的沙滩上踏沙戏水。麻木在工作上还骗自己他还小，还有很多机会。

未来

公司的希望是我们把事做好。明天如果不能适应公司的节奏，自然会被淘汰掉，不会顾及你之前多么卖命。

自己对未来的希望是：

公司对自己的需求不仅仅是肯卖命，而是有能力。未来可以投身到更有挑战的事情上去。

可以去想去的地方，不需要跟阿猫阿狗竞争，可以痛快地进行一番努力和较量。

希望自己朝目标一步一步努力下去，而不是沉浸在伪努力中。

#总结

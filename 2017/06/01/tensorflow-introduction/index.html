可视化

通常需要直观地看到训练的过程，此时就需要用到TensorBoard，分三步：

  • 代码中收集数据：
    • tf.summary.histogram
    • tf.summary.scalar
  • 写入文件
    • merged = tf.summary.merge_all()
    • writer = tf.summary.FileWriter("/tmp/board/", sess.graph)
    • result = sess.run(merged, feed_dict={xs: x_data, ys: y_data})
    • writer.add_summary(result, i)
  • 在浏览器上查看
    • tensorboard --logdir="/tmp/board/"

常用方法

常用的一些个性方法（下面这些我也不知道该怎么归类）：

  • tf.nn.conv2d：卷积
    • input：图像Tensor，格式为[batch, in_height, in_width, in_channels]
      • batch：训练时一个batch的数量
      • in_height：高度
      • in_width：宽度
      • in_channels：通道数
    • filter：卷积核，格式为[filter_height, filter_width, in_channels, out_channels]
      • filter_height：卷积核的高度
      • filter_width：卷积核的宽度
      • in_channels：通道数
      • out_channels：卷积核个数
    • strides：步长（需要指定每个维度的步长）
    • padding：卷积方式
      • SAME：
      • VALID：
    • use_cudnn_on_gpu：是否加速
  • tf.nn.max_pool：最大值池化
    - value：需要池化的输入，通常是【卷积】+【激活】处理后的维度
    - ksize：窗口，一般为[1, height, width, 1]
    - strides：同上
    - padding：同上
  • tf.nn.dropout：防止过拟合，一般用在全连接层
    • x：输入
    • keep_prob：神经元被选中的概率
    • noise_shape：一维Tensor，代表随机产生的“保留/丢弃”标志的Shape
    • seed：随机数种子
  • tf.truncated_normal：输出正太分布随机值
    • shape：输出的样子
    • mean：均值
    • stddev：标准差
    • dtype：类型
    • seed：随机种子
  • tf.matmul：矩阵乘法，常用来创建全连接

经常见到的激活函数包括（输入和输出维度相同）：

  • tf.nn.relu：max(features, 0)
  • tf.nn.relu6：min(max(features, 0), 6)
  • tf.nn.elu
  • tf.nn.sigmoid：y = 1 / (1 + exp(-x))
  • tf.nn.tanh
  • tf.nn.crelu
  • tf.nn.softplus：log(exp(features) + 1)
  • tf.nn.softsign
  • tf.nn.softmax：exp(logits) / reduce_sum(exp(logits), dim)

压缩函数系列（输出的维度会减少）：

  • tf.reduce_mean
  • tf.reduce_max
  • tf.reduce_sum
